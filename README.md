PingTools 网络 Ping 测试工具
一款基于 C# WPF 开发的 Windows 轻量批量 Ping 工具，用于工业现场、局域网设备连通性检测，实时统计延迟与丢包数据，支持日志持久化保存。
一、功能介绍
核心 Ping 检测
输入 IP / 域名一键启动持续 Ping 测试，实时输出每条响应记录
自动统计关键指标：
最大延迟、最小延迟、平均延迟（单位 ms）
发包总数、收包总数、丢包数量、丢包率百分比
日志显示控制
✅ 显示时间戳：每条记录附带精确时间，方便故障溯源
✅ 自动滚动：新日志自动下拉至底部，无需手动拖动
清屏按钮：一键清空全部输出日志
日志持久化
可选开启「日志保存」，自定义本地.txt存储路径
所有 Ping 记录完整写入文本文件，离线可复盘网络波动
勾选日志保存如果没有选择日志保存地址和日志名称，会默认保存地址为软件根目录，日志名称为地址栏内容。
二、界面说明
地址输入框：填写需要测试的 IP 地址、主机域名
开始按钮：绿色触发按钮，启动 / 停止 Ping 探测
数据统计栏：实时刷新延迟、发包、丢包统计数据
日志输出区域：展示每条 Ping 返回的详细结果
功能复选框：控制时间戳、自动滚动、日志保存开关
日志路径配置：自定义 txt 日志文件存放位置
三、使用教程
运行 PingTools.exe 打开程序
在「地址」输入框填写目标 IP（如 192.168.1.1）
按需勾选功能：
需要记录时间勾选「显示时间戳」
需要自动查看最新日志保持「自动滚动」勾选
需要留存记录勾选「日志保存」，并点击右侧按钮选择保存路径
点击绿色开始按钮，程序持续发送 Ping 包并实时展示数据
测试完成再次点击按钮停止，可点击「清屏」清空界面日志
四、适用场景
工业 PLC、交换机、工控机局域网连通性巡检
机房网络稳定性长时间压力测试
排查网络丢包、延迟波动故障
现场设备上线前连通性批量验证
网络故障日志留存、后期问题复盘
